# HTML Media Player 🎵🎬

This is a simple HTML project that plays Bengali songs and videos.

## 🎶 Features

- Bengali **audio songs** 🎧  
- Bengali **video songs** 🎥  
- Built using **pure HTML**

## 📁 How to Use

Just open the `index.html` file in any web browser and enjoy the media.

## 💡 Language

- **HTML**
